# TwitSplit

## Requirements

- iOS 10.0+ / macOS 10.13.1+
- Xcode 9.2+
- Swift 4.0+
- Cocoapod 1.4.0
## TestApp
- Download git repository
- Unzip
- Open Terminal
- Go to TwitSplit folder via Terminal
- Enter "pod install" in Terminnal
- Open .workspace
- Run and test app

## Unit testing
- Open .workspace
- Move to "Show the Test Navigatior" in left panel
- Clicl "Play" button in HomeViewModelTest
